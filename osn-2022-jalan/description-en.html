Description

Pak Dengklek is a foreman of road construction projects in his hometown. Each road consists of a gravel base paved with asphalt.

Pak Dengklek will buy the construction materials from $N$ shops, numbered from $1$ to $N$. Each shop only sells one kind of material. Shop $i$ sells the materials as follows:

  • If $T_i = 1$, then the shop sells gravel. One sack of gravel bought from this shop can build $A_i$ metres of gravel base for a road.
  • If $T_i = 2$, then the shop sells asphalt. One sack of asphalt bought from this shop can pave $A_i$ metres of gravel base.

Pak Dengklek has a unique method of building a road:

  1. Choose two integers $L$ and $R$ with $1 \le L \le R \le N$.
  2. Buy materials from shops $L$ to $R$ sequentially. In shop $L$, buy at least one sack of material. In each of the next shops, the number of sacks bought must be strictly greater than in the previous one.
  3. Build the gravel base of a road with all gravels bought.
  4. Pave the gravel base with the asphalt bought. If the amount of asphalt is insufficient to pave the entire road, the remaining road segment is left unpaved. If there is some remaining asphalt, it will be thrown away and not be usable anymore.
  5. Pak Dengklek must buy the materials from the shops in such a way as to minimize the length of the unpaved road segment.

Pak Dengklek will do $N$ road construction projects, numbered from $1$ to $N$. In the $j$-th project, Pak Dengklek will build $j$ roads. Pak Dengklek will build the $k$-th road in the $j$-th project according to the method above by choosing $L = k$ and $R = j$.

From all roads built in all projects, Pak Dengklek wants to know the total length of the unpaved road segments. Because this total length might be very large, Pak Dengklek only wants to know the value modulo $1\;000\;000\;007$. Help Pak Dengklek compute this value!

Constraints

  • $1 \le N \le 100\;000$
  • $1 \le A_i \le 100\;000$
  • $T_i = 1$ or $T_i = 2$

Subtasks

  1.   (6 points) Contains only the following test case:
    $N = 4$, $A = [3, 2, 2, 3]$, $T = [1, 2, 2, 1]$.
  2. (11 points) $N \le 400$, $T_i = 1$
  3.   (9 points) $N \le 3000$, $T_i = 1$
  4. (10 points) $T_i = 1$
  5. (21 points) $N \le 400$
  6.   (9 points) $N \le 3000$
  7. (17 points) There is no $i$ satisfying $T_i = 1$ and $T_{i+1} = 2$.
  8. (17 points) No additional constraints.

Input

The input is given with the following format: 

N
A1 A2 … AN
T1 T2 … TN

Output

Output a single line containing an integer representing the total length of the unpaved road segments modulo $1\;000\;000\;007$.

Sample Input 1

7
3 1 1 2 1 2 3
1 2 1 2 1 2 1

Sample Output 1

53
