Description

In Pak Dengklek's project, there are $N$ empty lands at the country border. The lands are numbered from $1$ to $N$ from west to east. Land $i$ directly borders land $i+1$ (for $1 \le i < N$). Land $i$ streches from west to east for:

  • $\displaystyle\frac{i+1}{2}$ metres, if $i$ is odd;
  • $N + 1 - \displaystyle\frac{i}{2}$ metres, if $i$ is even.

Pak Dengklek has $Q$ plans. In the $j$-th plan, Pak Dengklek wants to build a defence fortress with a length of exactly $X_j$ metres. Pak Dengklek can build a defence fortress on one or more consecutive lands from the west end of land $L$ to the east end of land $R$, for some pair of integers $L$ and $R$ with $1 \le L \le R \le N$.

For each plan, Pak Dengklek is curious about how many pairs $(L,R)$ that result in a defence fortress with the desired length. Help Pak Dengklek count the number of possible pairs $(L,R)$ in each plan!

Constraints

  • $1 \le N \le 10^9$
  • $1 \le Q \le 100\;000$
  • $1 \le X_j \le 10^{18}$

Subtasks

  1. (10 points) Contains only the following test case:
    $N = 8$, $Q = 5$, $X = [20, 21, 37, 6, 24]$.
  2. (12 points) $N \le 400$, $Q \le 400$
  3. (16 points) $N \le 1000$
  4. (24 points) $N \le 100\;000$, $Q = 1$
  5. (23 points) $N \le 100\;000$
  6. (15 points) No additional constraints.

To build a defence fortress with a length of exactly $6$ metres, the pairs $(L,R)$ that can be chosen are $(1,2)$ and $(3,4)$.
